Pantry Warrior
What
Pantry Warrior is a 2D platformer with a focus on vertical progress. Play as a bug inflitrating the rat pantry in their journey to collect crumbs and feed their colony. Defeat enemies and level up to unlock new skills and challedge harder bosses as you climb through the levels!
Why
This is an on going passion project. I have always been interested in gaming, and this was my way of staying productive, learning more about game design, and creating an outlet where I could create art all in one.
How
This is developed on the godot game engine using gdscript. Art assets are drawn in Aesprite.
UCollect
What
UCollect is my spin on a URL shortening tool with a primary focus of organization. The unique mechanic of this tool is to allow anyone to create and maintain collections of URLs. A collection is a group of shortened URLs that can be updated by the creator and associates.
Why
The main use case of this tool was designed to solve a problem in my life: cluttered group chats. Examples: Agoda links from trip planning, Amazon links from wishlists, IEEE links for future Citation Machine usage... These massive text blocks are hard to keep track of, and ruin the chat for everyone. Now, a single link is all you need.
How
One of the primary goals in this project was to learn new technologies. ExpressJS and MongoDB where used to store and manage the data and React was used for the UI.
Mango
What
Mango is a cross between an old math game that I played and Bingo. The board and win conditions are the same of that of Bingo, but to get to the squares on the board, you must created them using the numbers cards given to you. There are multiple configurations for: board size, number of cards distributed, how cards are distributed and win condition. Once you get comfortable, challedge your friend online!
Why
This is a game I have had in mind for a long time. I grew up playing a game I called 24, where after revealing 4 playing cards, all players would attempt to be the first to create the number 24 using only basic operations. The goal when approaching this project was to learn something about responsive UI and make a game which I could play with my family.
How
The webapp is created using Angular and Typescript. Server is hosted using ExpressJS.
Essay Tools
What
A suite of tools to help students! Currently includes a word frequency parser and a word counter for quotes. In the frequency parser, you configure words to ignore, sentence enders, word enders and the number of sentences to check at once. Upon submitting your esssay, the tool will highlight all duplicate words used within the configured sentence count. The word counter is a tool to get a word count, ignoring quotes.
Why
I have a kid sister who couldn't find these tools online. Seemed pretty fun and easy to throw together a script for them and it helps her out too.
How
React + Typescript for UI. Uvicorn + FastAPI for backend server and API framework.
